# Drafts / Borradores

Esta carpeta contiene los posts originales del tema Retypeset que fueron movidos aquí para evitar que se rendericen en el sitio.

Los archivos están organizados de la siguiente manera:

## Estructura

- `Universal Post.md` - Post de ejemplo universal
- `examples/` - Posts de demostración de funcionalidades
  - Posts de demostración de KaTeX (matemáticas)
  - Posts literarios en chino/japonés con traducciones
- `guides/` - Guías del tema
  - Guías de características extendidas de Markdown
  - Guías de estilo y esquemas de color
  - Documentación del tema
- `_images/` - Imágenes de los posts de ejemplo

## Uso

Estos archivos pueden servir como:
- Referencia para ver cómo usar las funcionalidades del tema
- Plantillas para crear nuevos posts
- Ejemplos de sintaxis de Markdown extendida
- Demostraciones de internacionalización

Para usar cualquiera de estos posts, simplemente cópialos de vuelta a `src/content/posts/` y modifica el contenido según necesites.
